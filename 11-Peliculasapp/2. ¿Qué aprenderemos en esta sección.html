El Objetivo principal:

El objetivo de esta sección es que seamos capaces de aplicar lo aprendido a lo largo del curso para poder crear la aplicación de películas que veremos a continuación, no tiene que verse igual, pero la funcionalidad si la debe de tener.

No se preocupen, que se explicará como utilizar las APIs que ofrece la pagina Themoviedb, para que podamos consumir sus servicios.


TAREA:

Si hemos seguido el curso, para estas alturas ya deberíamos de ser capaces de realizar esta tarea con un mínimo de dificultad.

La idea es crear una aplicación que nos permita mostrar una página Home con la información de las películas actuales o populares, otra pantalla donde podamos observar información adicional de la película seleccionada y aparte poder tener un buscador para poder encontrar otras películas.

Para obtener la información utilizaremos el servicio que ofrece la gente de https://www.themoviedb.org/, que tienen toda la información necesaria para realizar este trabajo.
